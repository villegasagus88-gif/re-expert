# Caso: Galpón logístico clase A AMBA

## TL;DR
- 15.000 m² techados clase A para operador logístico / ecommerce.
- Modalidad: Build to Suit (BTS) con contrato pre-firmado triple net 10 años.
- Salida: venta a fondo industrial o securitización.
- Cap rate objetivo a estabilización: 9-11%.

## 1. Snapshot

| Variable | Valor |
|---|---|
| Localización | Mercado Central / Esteban Echeverría / Ezeiza / ramal Pilar (Cruce con accesos clave) |
| Lote | 30.000-40.000 m² |
| Techado | 15.000 m² |
| Altura libre | 12 m |
| Docks | 25-30 |
| Plazo obra | 12-18 meses |
| Modalidad | Build to Suit con inquilino ancla |

## 2. Stakeholders

- **Developer industrial**: experiencia en galpones clase A.
- **Inquilino ancla**: operador 3PL, ecommerce, retailer (Mercado Libre, Amazon, Andreani, Loginter, Carrefour, etc.).
- **Inversor institucional**: opcional, para co-equity.
- **Constructora especializada** en logístico: estructuras metálicas + losas industriales.
- **Project manager**.

## 3. Estructura financiera

### 3.1 Costos estimados (USD)

| Rubro | USD | % |
|---|---|---|
| Tierra | 4.000.000 | 25% |
| Costo directo obra (USD 600/m²) | 9.000.000 | 56% |
| Infraestructura (accesos, playón) | 1.000.000 | 6% |
| Honorarios + dirección | 600.000 | 4% |
| Permisos | 200.000 | 1% |
| Project management | 300.000 | 2% |
| Contingencia | 1.000.000 | 6% |
| **Total** | **16.000.000** | **100%** |

### 3.2 Renta y NOI

- Renta clase A: USD 7-10/m²/mes triple net.
- Ingreso bruto: ~USD 1.500.000/año.
- Opex no cobrable (gestión + reposición): mínimo en TNN.
- **NOI**: ~USD 1.400.000/año.

### 3.3 Valuación a salida

- Cap rate 10%: USD 14.000.000.
- Cap rate 9%: USD 15.500.000.

### 3.4 Yield on cost

- ~8,75% — comparativamente alto vs. oficinas / BTR.

### 3.5 Financiación

- 60% equity / 40% deuda construcción.
- BTS firmado pre-obra reduce riesgo y abarata deuda.

## 4. Comercialización pre-obra

### 4.1 BTS
- Contrato pre-firmado con inquilino antes de iniciar obra.
- Specs personalizadas al usuario (alto, docks, racks, oficinas anexas).
- Garantías exigidas al inquilino: fianza corporativa + seguro caución 12 meses.

### 4.2 Multi-tenant (alternativa)
- Si no hay BTS: dividir en módulos.
- Comercialización más lenta.

## 5. Aspectos legales / tributarios

### 5.1 Forma jurídica
- SPE (SA / SAS) o fideicomiso de inversión.
- IIGG sobre rentas.

### 5.2 Impuestos
- IVA: alquiler comercial gravado 21% si supera mínimo + opta por gravado.
- IIBB jurisdiccional.
- Inmobiliario provincial / municipal.

### 5.3 Regulatorio
- Permiso municipal industrial.
- Habilitación ambiental (estudio de impacto si aplica).
- Bomberos.
- Habilitación comercial / industrial.
- Si almacena cargas peligrosas: registros específicos.

### 5.4 Macro AR
- Logística es segmento más estable: demanda por ecommerce + canales modernos.
- Renta en USD posible por demanda de multinacionales.

## 6. Riesgos y mitigación

| Riesgo | Mitigación |
|---|---|
| Pérdida del ancla | BTS con plazo 10 años + garantías + cláusulas de salida |
| Atraso de obra | GMP + caución cumplimiento + penalty si supera plazo |
| Costo construcción (acero, hormigón) | Compras anticipadas + cláusulas redeterminación |
| Macro (importaciones, ecommerce baja) | Diversificación de inquilinos potenciales |
| Ambiental | Estudio previo + seguros |
| Acceso vial / infraestructura | Selección de lote estratégica + permisos |

## 7. Indicadores objetivo

| KPI | Valor objetivo |
|---|---|
| TIR equity | 15-20% USD |
| Yield on cost | ~8-9% |
| Cap rate salida | 9-11% |
| Vacancia (post-ancla) | < 5% |
| WALT | ≥ 7 años a la venta |

## 8. Tipos de galpones por demanda AR

| Tipo | Demanda | Renta USD/m²/mes |
|---|---|---|
| Clase A (12m alto, racks, docks) | Ecommerce, 3PL multinacional | 7-10 |
| Clase B | Retail local, fabricas medianas | 4-6 |
| Clase C | Talleres, pequeñas industrias | 2-4 |
| Build to Suit especializado | Cargas peligrosas, frio | Premium variable |

## 9. Lecciones / pitfalls

- Sin ancla pre-firmada → riesgo muy alto.
- Specs no negociadas → reformas post-entrega.
- Lote con acceso vial limitado → no es clase A real.
- Subestimar infraestructura (energía, fibra, agua).
- Permisos ambientales tardíos.
- Contratos no triple net → opex erosiona NOI.
- Olvidar habilitación bomberos / SRT.

## 10. Reglas operativas para el chat

- **Estable:** estructura BTS, clase A vs B/C, lógica triple net.
- **🔴 Volátil:** USD/m² obra, rentas, cap rates → mercado al día.
- **Sensible:** logística requiere DD especializada (acceso vial, ambiental, energía).
