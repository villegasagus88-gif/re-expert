# Caso: BTR / Multifamily suburbano AMBA

## TL;DR
- 120 unidades en zona norte AMBA, diseñadas exclusivamente para renta institucional.
- No venta retail: hold + operación + eventual exit a fondo core.
- Cap rate objetivo a estabilización: 7-9%.
- Producto incipiente en AR (vs. mercado maduro EEUU / Europa).

## 1. Snapshot

| Variable | Valor |
|---|---|
| Localización | Pilar / Nordelta / Vicente López / San Isidro |
| Lote | 4.000-8.000 m² |
| Programa | 120 unidades + amenities resort |
| Mix | 30% 1amb, 50% 2amb, 20% 3amb |
| Amenities | piscina, gym, coworking, gastronomía, pet area, lavandería compartida |
| Plazo obra | 30-36 meses |
| Plazo estabilización | 12-18 meses |
| Modalidad | Single ownership institucional (no PH para venta) |

## 2. Stakeholders

- **Developer especializado en BTR**: experiencia en operación de renta.
- **Inversor institucional**: fondo internacional, family office, fondo de pensión.
- **Operador profesional** (property management).
- **Constructora top tier**: GMP.
- **Marca / branding** (a veces marca operadora reconocida).

## 3. Estructura financiera

### 3.1 Costos estimados (USD)

| Rubro | USD | % |
|---|---|---|
| Tierra | 4.000.000 | 16% |
| Costo directo obra (USD 1.500/m², 9.000 m²) | 13.500.000 | 54% |
| Honorarios + dirección | 1.200.000 | 5% |
| Permisos + tasas | 400.000 | 2% |
| Project management | 600.000 | 2% |
| FF&E (mobiliario, amenities) | 1.500.000 | 6% |
| Soft costs + financieros | 1.300.000 | 5% |
| Contingencia | 2.500.000 | 10% |
| **Total** | **25.000.000** | **100%** |

### 3.2 Operación estabilizada

- Renta promedio: USD 800-1.500/mes/unidad (zona y mix).
- Ingreso bruto: ~USD 1.500.000/año.
- Vacancia: 8-12%.
- Opex: 25-35% del ingreso bruto.
- **NOI**: ~USD 900.000/año.

### 3.3 Valuación a salida

- Cap rate 8%: USD 11.250.000.
- Cap rate 9%: USD 10.000.000.

Nota: a estos precios el yield on cost es bajo y solo cierra si hay apreciación de cap rate (compresión por institucionalización del producto). Modelo apuesta a tendencia.

### 3.4 Financiación

- Equity 100% institucional (rare debt en AR para BTR).
- Posible deuda en USD a estabilización.

## 4. Lease-up

### 4.1 Pricing
- USD 800-1.500/mes según unidad.
- Contratos: 12-24 meses con renovación.
- Step-up por CPI USA o índice.

### 4.2 Carencia
- 1-2 meses gratis en lease-up inicial.

### 4.3 Servicios incluidos / extra
- Expensas incluidas (operador absorbe).
- Internet + cable incluidos como upsell.

## 5. Tributario y regulatorio

### 5.1 Forma jurídica
- SPE (SA o SAS) o fideicomiso de inversión.
- IIGG sobre rentas + ganancias.
- IVA: alquiler vivienda exento (régimen general); reducciones según uso.
- Bienes Personales: estructura para minimizar (sociedad).

### 5.2 Regulatorio
- Permiso obra municipio.
- Habilitaciones del operador.
- Locaciones: marco CCyCN + Ley de Alquileres vigente.

### 5.3 Macro AR
- Renta residencial AR tradicionalmente baja (cap rate negativo en pesos por inflación).
- BTR exige renta en USD para tener sentido.

## 6. Riesgos y mitigación

| Riesgo | Mitigación |
|---|---|
| Producto incipiente, mercado no maduro | Branding fuerte + amenities |
| Marco regulatorio alquileres cambiante | Plazos contractuales y diversificación |
| Vacancia mayor a modelada | Operador profesional + marketing digital |
| Cap rate no comprime | Hold extendido + cash flow positivo |
| Devaluación / inflación | Contratos en USD o CER |
| Cobranza | Garantías exigidas a inquilinos |

## 7. Indicadores objetivo

| KPI | Valor objetivo |
|---|---|
| TIR equity | 10-14% USD |
| Yield on cost estabilizado | ~6% (bajo) |
| Cap rate salida | 7-9% |
| Vacancia estabilizada | ≤ 10% |
| Opex ratio | ≤ 30% |
| WALT contractos | ~1,5 años |

## 8. Comparativa con venta retail

| Variable | BTR | Venta retail |
|---|---|---|
| Cash flow | Recurrente | Único |
| Inversión | Pesada y larga | Cíclica |
| Inversor típico | Institucional | Particular + institucional |
| Riesgo de mercado | Renta + cap rate | Precio + ritmo |
| Salida | Venta block | Múltiples ventas |

## 9. Lecciones / pitfalls

- Subestimar costo de operación (amenities + servicios).
- Diseño no flexible — habrá unidades que no se renten.
- Cobranza en AR es difícil sin garantías sólidas.
- Marco regulatorio de alquileres puede cambiar.
- Sin operador profesional: erosión del valor.
- Esperar cap rates de EEUU en AR es ingenuo.
- Asumir 0% vacancia.

## 10. Reglas operativas para el chat

- **Estable:** lógica BTR, KPIs, comparativa con venta.
- **🔴 Volátil:** rentas USD, cap rates, vacancia → mercado AR específico.
- **Sensible:** BTR institucional requiere estructura legal/tributaria cuidada.
