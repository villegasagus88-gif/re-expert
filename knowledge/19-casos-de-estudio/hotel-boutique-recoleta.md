# Caso: Hotel boutique 25 llaves Recoleta / Palermo

## TL;DR
- Hotel boutique 4-5 estrellas, 25 llaves, en barrio premium CABA.
- Reuso adaptativo de petit hotel / casona o construcción nueva pequeña.
- Estructura: propiedad + operador profesional bajo HMA (Hotel Management Agreement).
- Cap rate objetivo: 10-12% (premium por riesgo operativo).

## 1. Snapshot

| Variable | Valor |
|---|---|
| Localización | Recoleta / Palermo Soho / Belgrano |
| Programa | 25 habitaciones + restó / bar + amenities |
| Modalidad construcción | Reuso adaptativo + ampliación |
| Plazo obra | 18-24 meses |
| Pre-apertura | 3-6 meses |
| Operación | Operador profesional bajo HMA |

## 2. Stakeholders

- **Propietario / developer**: persona / sociedad / fondo.
- **Operador hotelero**: marca boutique (independiente) o cadena boutique (Hyatt Caption, IHG Vignette, Marriott Tribute, Accor MGallery).
- **Arquitecto especializado** en hotelería.
- **Diseñador de interiores**.
- **Consultor hotelero** para feasibility + pre-apertura.

## 3. Estructura financiera

### 3.1 Costos estimados (USD)

| Rubro | USD | % |
|---|---|---|
| Adquisición edificio existente o tierra | 3.500.000 | 35% |
| Obra civil + restauración | 3.000.000 | 30% |
| FF&E + OS&E (mobiliario + equipamiento operativo) | 1.500.000 | 15% |
| Cocina + restó + bar | 500.000 | 5% |
| Honorarios + dirección | 400.000 | 4% |
| Permisos + habilitaciones | 200.000 | 2% |
| Branding + marketing pre-apertura | 300.000 | 3% |
| Capital de trabajo pre-apertura | 300.000 | 3% |
| Contingencia | 300.000 | 3% |
| **Total** | **10.000.000** | **100%** |

### 3.2 Operación estabilizada (año 3-4)

- ADR (Average Daily Rate): USD 200-300.
- Occupancy: 65-75%.
- **RevPAR**: USD 130-220.
- Ingreso anual: ADR × occupancy × 25 × 365 ≈ USD 1.500.000-2.000.000.
- Departamental contribution (después de COGS): ~50-60%.
- Gastos no distribuidos + reservas FF&E: ~25-30%.
- **GOP**: ~25-35% del ingreso.
- Honorarios management (base + incentive): 3-5% + 8-10% GOP.
- **NOI**: ~USD 350.000-500.000.

### 3.3 Valuación

- Cap rate hotelería 10-12% en AR: USD 3.500.000-5.000.000 (sub-óptimo en este modelo).
- Alternativa: múltiplos por llave (USD/llave): USD 250-400k para boutique premium → USD 6-10M.

### 3.4 Financiación

- 50-70% equity + deuda.
- Hotelería: difícil financiación bancaria local sin track record.

## 4. Operación

### 4.1 HMA (Hotel Management Agreement)
- Plazo: 10-20 años + renovación.
- Operador opera con marca + estándares.
- Owner paga base fee + incentive fee.
- Reportes mensuales + presupuesto anual aprobado.

### 4.2 Franchise vs operador propio
- Franchise: marca pero owner opera.
- Operador propio: si owner tiene know-how (raro en proyectos boutique).

### 4.3 Pre-apertura
- Hiring + training equipo.
- Plan de marketing + ventas + revenue management.
- Soft opening + grand opening.

## 5. Mercado AR

### 5.1 Sub-segmentos
- Lifestyle / boutique: ADR USD 200-400.
- Luxury (Park Hyatt, Four Seasons, Alvear): ADR USD 500-1.500.
- Upscale: USD 150-250.
- Midscale: USD 80-150.

### 5.2 Demanda
- Turismo extranjero (USD).
- Negocios + corporate.
- Bleisure post-pandemia creciente.
- Estacionalidad: pico Nov-Mar y eventos.

### 5.3 Cepo y FX
- Turismo extranjero paga en USD (vía pasarela o transferencia).
- Ingresos pueden estar en dólar blue informalmente o canalizados vía MEP.
- Compliance estricto vía PSAV / banco.

## 6. Aspectos legales / tributarios

### 6.1 Forma jurídica
- SA / SAS.
- IIGG.
- IVA: 10,5% sobre alojamiento (régimen reducido).
- IIBB.
- Tasas municipales + ABL.

### 6.2 Habilitaciones
- Habilitación hotelera (Ministerio Turismo + GCBA).
- Categorización (estrellas).
- Bromatología cocina.
- Bomberos.
- Música funcional / SADAIC / AADI-CAPIF.

### 6.3 Marca registrada
- INPI: registro de marca propia.
- Si franchise: contrato + tasas.

### 6.4 Laboral
- Convenio UTHGRA (gastronomía y hotelería).
- Régimen propio + propinas.

## 7. Riesgos y mitigación

| Riesgo | Mitigación |
|---|---|
| Macro AR (turismo cae) | Diversificación mercados emisores + dynamic pricing |
| FX volatilidad | Pricing en USD + cobros multimoneda |
| Operador subperforma | KPIs + cláusulas terminación |
| Costos operativos altos (sueldos UTHGRA) | Operación lean + tecnología |
| Estacionalidad fuerte | Eventos + corporate + grupos |
| Cambio gustos / OTA disrupción | Branding + marketing digital + canal directo |
| Reuso adaptativo: sorpresas obra | Due diligence técnica previa + contingencia |

## 8. Indicadores objetivo

| KPI | Valor objetivo |
|---|---|
| TIR equity | 12-18% USD |
| Cap rate salida | 10-12% |
| Occupancy estabilizado | ≥ 65% |
| ADR estabilizado | USD 200-300 |
| GOP margin | ≥ 30% |
| Rampa estabilización | Año 3-4 |

## 9. Lecciones / pitfalls

- Subestimar capital de trabajo pre-apertura.
- Operador sin track record local.
- HMA con cláusulas pro-operador difíciles de salir.
- Falta de feasibility study riguroso.
- Sobre-invertir en FF&E sin ROI.
- Compliance laboral UTHGRA mal gestionado.
- Cobros en negro / cueva → riesgo UIF + AFIP severo.
- Estacionalidad mal calculada en pro-forma.

## 10. Reglas operativas para el chat

- **Estable:** estructura, KPIs, lógica de hotel boutique.
- **🔴 Volátil:** ADR, occupancy, cap rate, USD/llave → mercado y temporada.
- **Sensible:** hotelería tiene operación + RE; requiere doble expertise.
