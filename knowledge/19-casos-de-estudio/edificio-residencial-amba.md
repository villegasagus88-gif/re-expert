# Caso: Edificio residencial premium AMBA

## TL;DR
- Proyecto modelo: edificio de 30 unidades en barrio premium CABA / GBA Norte.
- Estructura: fideicomiso al costo + inversores ancla + venta pozo.
- Cronograma 30 meses obra + 6 meses pre-obra + 6 meses comercialización.
- TIR objetivo developer: 18-25% USD.

## 1. Snapshot

| Variable | Valor |
|---|---|
| Localización | Barrio premium CABA (Palermo / Belgrano / Caballito alto) |
| Lote | 600 m² |
| FOT permitido | 3,5 → ~2.100 m² construibles |
| Programa | 30 unidades: 8x1amb, 12x2amb, 8x3amb, 2x dúplex |
| Amenities | SUM, gym, piscina, parrilla, coworking |
| Cocheras | 25 |
| Plazo obra | 30 meses |
| Modalidad | Fideicomiso al costo + venta pozo |

## 2. Stakeholders

- **Developer**: PyME local con 3-5 proyectos previos. Fiduciante + administrador.
- **Estudio de arquitectura**: contratado externamente.
- **Constructora**: por administración con presupuesto guía.
- **Director técnico** matriculado CPAU.
- **Inversores ancla**: 5-8 invierten 10-30% del proyecto desde el inicio.
- **Compradores pozo**: 22-25 unidades en venta pre-obra y durante obra.
- **Escribano**: prepara fideicomiso + escrituras.
- **Inmobiliaria**: comercialización exclusiva.

## 3. Estructura financiera

### 3.1 Costos estimados (USD)

| Rubro | USD | % |
|---|---|---|
| Tierra | 2.000.000 | 22% |
| Costo directo obra (USD 1.500/m²) | 4.500.000 | 50% |
| Honorarios + dirección | 350.000 | 4% |
| Permisos + tasas + COT | 200.000 | 2% |
| Gerenciamiento (5%) | 250.000 | 3% |
| Comercialización (4%) | 480.000 | 5% |
| Financieros + soft costs | 320.000 | 4% |
| Contingencia (10%) | 900.000 | 10% |
| **Total** | **9.000.000** | **100%** |

### 3.2 Ingresos estimados

- Promedio venta: USD 3.500/m² (área propia + 30% balcones/amenities).
- Superficie vendible: ~1.700 m² propios + cocheras.
- Ingreso bruto: ~USD 12.000.000.
- Margen bruto: ~USD 3.000.000 (~25%).

### 3.3 Financiación

- 25% inversores ancla → USD 2.250.000 en pre-obra.
- 75% ventas pozo → USD 6.750.000 escalonado.
- Sin deuda bancaria.

## 4. Cronograma y curva S

| Mes | Hito |
|---|---|
| 0-6 | Pre-obra: permisos, plano municipal, fideicomiso, comercialización pre-obra |
| 6-12 | Excavación + cimientos + estructura niveles bajos. Inversores aportan. Venta pozo 30% |
| 12-24 | Estructura completa + mampostería + instalaciones. Venta pozo 50% acumulado |
| 24-30 | Terminaciones + amenities. Venta pozo 80% acumulado |
| 30-32 | Final de obra, certificado, subdivisión PH |
| 32-36 | Escrituración + entregas |

Curva S de costos: típica con pico mes 18.

## 5. Comercialización

- **Pre-lanzamiento**: comunicación + brochure + render + show room virtual.
- **Lanzamiento** mes -3 a 0: descuento 8-12% sobre precio entrega.
- **Pozo temprano** (0-12 meses obra): descuento 5-8%.
- **Pozo medio** (12-24): descuento 2-5%.
- **Pre-entrega**: precio lista.
- Cuotas en pesos ajustables (índice CAC o USD).
- Reserva 5% + boleto 20% + cuotas + saldo escritura.

## 6. Aspectos legales / tributarios / regulatorios

### 6.1 Forma jurídica
- Fideicomiso de administración (Ley 26.994 art. 1.666).
- Fiduciante: developer + inversores.
- Beneficiarios: inversores + compradores que adhieren.

### 6.2 Impuestos
- IVA: 10,5% sobre obra nueva vivienda. Crédito fiscal por compras.
- IIGG: tributa el fideicomiso o se atribuye según contrato.
- Bienes Personales: cada beneficiario por su participación.
- ITI o IIGG cedular 15%: vendedores personas humanas.
- Sellos CABA: 1% boleto (con reducciones a vivienda única).
- ABL + ARBA cuando aplique.

### 6.3 Regulatorio
- Permiso de obra Dirección General de Registro Obras y Catastro (CABA) o municipio.
- COT 2.069 y demás certificaciones.
- Plano de subdivisión PH.
- Reglamento de copropiedad.
- UIF: cada compra > umbral requiere DDC.

### 6.4 Seguros
- TRC durante obra.
- RC obra.
- ART de personal.
- Caución de cumplimiento del constructor.

## 7. Riesgos y mitigación

| Riesgo | Mitigación |
|---|---|
| Desvío de costos por inflación | Contratos en USD o CAC, contingencia 10% |
| Cepo cambiario / brecha | Pricing USD, cobro escalonado |
| Caída de demanda | Pricing dinámico + cuotas flexibles |
| Atraso de obra | Constructora con caución cumplimiento |
| Vicios de obra | TRC + garantía vicios redhibitorios |
| Cambio normativo | Permiso firme antes de inversión grande |
| UIF | KYC riguroso a inversores + compradores |

## 8. Indicadores objetivo

| KPI | Valor objetivo |
|---|---|
| TIR developer | 18-25% USD |
| TIR inversor ancla | 12-15% USD |
| Margen bruto | 22-28% |
| Payback developer | ~3 años desde inicio |
| % vendido a entrega | ≥ 85% |
| Desvío costo vs presupuesto | ≤ 8% |

## 9. Lecciones / pitfalls

- Subestimar contingencia → desviación devora margen.
- Sobre-comercialización temprana → no se aprovecha curva de precios.
- Mezclar fondos del fideicomiso con fondos del developer (riesgo legal).
- No documentar origen de fondos de compradores (riesgo UIF).
- Constructor sin caución → atraso sin solución.
- Inversores ancla mal alineados → conflicto.
- Plano de subdivisión PH tardío → atraso de escrituras + costos financieros.
- Comercializar con precio USD billete sin claridad sobre canal (MEP / MULC / billete).
