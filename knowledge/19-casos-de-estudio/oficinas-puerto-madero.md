# Caso: Torre de oficinas A+ Puerto Madero

## TL;DR
- Edificio de oficinas A+ con certificación LEED Silver, 18.000 m² rentables.
- Estructura: fideicomiso financiero + inversor institucional ancla + reaseguro internacional.
- Modelo: desarrollo + lease-up + estabilización + venta a fondo core.
- Cap rate objetivo a la salida: 8-10%.

## 1. Snapshot

| Variable | Valor |
|---|---|
| Localización | Puerto Madero — dique 2 / 3 |
| Lote | 2.500 m² |
| Programa | 18.000 m² rentables, 12 plantas tipo de 1.500 m² + 2 niveles amenities |
| Cocheras | 220 |
| Certificación | LEED Silver |
| Plazo obra | 36 meses |
| Plazo lease-up | 18 meses post-entrega |
| Plazo total a venta | ~6 años |

## 2. Stakeholders

- **Developer**: empresa AR con track record corporativo.
- **Inversor institucional ancla**: fondo internacional o family office (40-60%).
- **Fiduciario financiero**: banco / sociedad fiduciaria registrada CNV.
- **Constructora top tier**: contrato GMP (Guaranteed Maximum Price).
- **Arquitectura internacional + local**: estudio AR asociado con firma internacional.
- **Project manager** independiente.
- **Property manager / asset manager** para etapa operativa.
- **Broker corporativo**: CBRE / Cushman / Colliers / JLL para lease-up.

## 3. Estructura financiera

### 3.1 Costos estimados (USD)

| Rubro | USD | % |
|---|---|---|
| Tierra | 25.000.000 | 25% |
| Costo directo obra (USD 2.500/m²) | 45.000.000 | 45% |
| Honorarios + dirección | 4.500.000 | 4,5% |
| Permisos + tasas | 1.500.000 | 1,5% |
| Project & cost management | 2.500.000 | 2,5% |
| Soft costs + financieros | 6.000.000 | 6% |
| Comercialización + lease-up | 3.500.000 | 3,5% |
| Contingencia (12%) | 12.000.000 | 12% |
| **Total** | **100.000.000** | **100%** |

### 3.2 Ingresos a estabilización

- Renta promedio: USD 30/m²/mes triple net.
- NOI anual estabilizado: ~USD 5.500.000 (después de vacancia 8% + opex no cobrable + gestión).
- Valor a cap rate 9%: ~USD 60.000.000 (sub-óptimo).

### 3.3 Estructura financiera

- 50% equity (developer 20% + institucional 80% del equity = 10% / 40% del total).
- 50% deuda construcción (USD 50M, banco internacional + reaseguro).
- A estabilización: refinanciación con deuda largo plazo o venta.

### 3.4 Estrategia de salida

- **A**: venta a fondo core internacional → cap rate 8-9%.
- **B**: securitización vía fideicomiso financiero (FF) en BYMA.
- **C**: hold + renta operativa.

## 4. Cronograma y curva S

| Mes | Hito |
|---|---|
| 0-12 | Pre-obra, fideicomiso, permisos, lanzamiento estructuración |
| 12-48 | Obra 36 meses, curva S clásica |
| 48-66 | Lease-up: 18 meses, target 85-90% ocupación |
| 66-72 | Estabilización + DD + transacción venta |

## 5. Lease-up

### 5.1 Pricing
- Listing: USD 30/m²/mes.
- Carencias: 2-4 meses al inicio para anchor tenants.
- Step-up: incrementos anuales por CPI USA o índice predefinido.

### 5.2 Pliego típico de contrato
- Plazo 5-10 años.
- Triple net (TNN): inquilino paga impuestos + seguros + expensas.
- Garantías: fianza + seguro caución 6-12 meses.

### 5.3 Anchor tenants
- Buscar 1-2 que ocupen 25-40% (banco, multinacional, consultora).
- Reduce riesgo de comercialización.

## 6. Aspectos legales / tributarios

### 6.1 Forma jurídica
- Fideicomiso financiero (CNV Ley 24.441 / 26.994).
- VDF + CP emitidos en BYMA si hay institucional.
- Sociedad de propósito específico SPE.

### 6.2 Impuestos
- IVA: oficinas tributan 21% (no la reducción de vivienda).
- IIGG: sociedad o fideicomiso.
- IIBB: ingresos brutos jurisdiccionales (CABA Convenio Multilateral).
- Bienes Personales: si hay participación de personas humanas.
- Sellos boleto: CABA.

### 6.3 Regulatorio
- Permiso CABA + Corporación Puerto Madero (planeamiento especial).
- LEED: documentación + certificación USGBC.
- Habilitación comercial post-entrega.

### 6.4 CNV
- Si emite FF: prospecto + DD + Calificadora + Audit.
- Reportes trimestrales.

### 6.5 Cambiario
- Capital extranjero: ingreso vía MULC (Ley 27.541 + Decreto 609/2019).
- Repatriación de utilidades: análisis al momento.

## 7. Riesgos y mitigación

| Riesgo | Mitigación |
|---|---|
| Cepo cambiario para inversor extranjero | Estructuración legal previa + asesor cambiario |
| Demanda corporativa (post-pandemia + híbrido) | Diseño flexible + amenities + sostenibilidad |
| Vacancia mayor a la modelada | Anchor tenant pre-obra + descuentos iniciales |
| Tasa de interés alta (deuda construcción) | Hedge si aplicable + estructura híbrida |
| Atraso obra | GMP + caución cumplimiento |
| Sub-mercado se enfría | Diseño multi-tenant convertible |
| Macro AR (recesión, devaluación) | Costos parcialmente en USD, renta en USD |

## 8. Indicadores objetivo

| KPI | Valor objetivo |
|---|---|
| TIR equity | 14-18% USD |
| TIR institucional | 12-15% USD |
| Cap rate salida | 8-10% |
| NOI / costo | ~5,5% (yield on cost a estabilización) |
| Lease-up time | ≤ 18 meses |
| WALT (vida promedio contratos) | ≥ 5 años a la venta |

## 9. Lecciones / pitfalls

- Subestimar lease-up post-pandemia → menor renta efectiva.
- No anclar inquilino pre-obra → riesgo comercialización.
- Diseño rígido no permite reconvertir a otro uso.
- Falta certificación LEED → menor demanda corporativa premium.
- Cambiar parámetros financieros midway sin acuerdo institucional → conflicto.
- Permisos sin firmeza al cierre financiero → ofreció timing.
- Ignorar zoning especial Puerto Madero.

## 10. Reglas operativas para el chat

- **Estable:** estructura, hitos, lógica institucional.
- **🔴 Volátil:** USD/m², cap rate, tasas → mercado al día.
- **Sensible:** proyecto institucional requiere mesa completa de asesores.
