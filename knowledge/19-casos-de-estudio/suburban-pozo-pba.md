# Caso: Barrio cerrado / club de campo PBA

## TL;DR
- Desarrollo de 200 lotes + 50 casas pozo en PBA (zona norte / oeste).
- Plazo extendido: 7-10 años desde compra de tierra a último cierre.
- Estructura mixta: venta de lotes + venta de casas pozo + amenities club.
- TIR objetivo developer: 18-25% USD en plazo largo.

## 1. Snapshot

| Variable | Valor |
|---|---|
| Localización | Pilar / Escobar / Tigre / Cañuelas / Brandsen |
| Superficie total | 120 hectáreas |
| Lotes vendibles | 200 (promedio 800-1.200 m²) |
| Casas pozo (alternativa o complemento) | 50 |
| Amenities | club house, piscina, canchas, lago, áreas verdes |
| Plazo desarrollo | 7-10 años |
| Modalidad | Venta de lotes + venta de casas + administración del club |

## 2. Stakeholders

- **Developer**: empresa con experiencia en desarrollos suburbanos.
- **Sociedad civil / asociación civil del club**: administra una vez vendido.
- **Constructora**: opcional, si developer también construye casas.
- **Inmobiliaria**: comercialización exclusiva o multi-broker.
- **Estudio jurídico** especializado en desarrollos urbanísticos.
- **Agrimensor** para mensura y subdivisión.
- **Municipio + Provincia** (organismos de aprobación).

## 3. Estructura financiera

### 3.1 Costos estimados (USD)

| Rubro | USD | % |
|---|---|---|
| Tierra (120 ha) | 8.000.000 | 25% |
| Master plan + ingeniería | 1.000.000 | 3% |
| Movimiento de suelos + nivelación | 3.500.000 | 11% |
| Infraestructura (calles, electricidad, agua, cloacas, gas, fibra) | 12.000.000 | 38% |
| Amenities (club house, piscinas, canchas) | 3.000.000 | 9% |
| Permisos + estudios ambientales + tasas | 1.500.000 | 5% |
| Comercialización + marketing | 1.500.000 | 5% |
| Honorarios + gerenciamiento | 1.500.000 | 4% |
| **Total infra + amenities** | **32.000.000** | **100%** |

(Construcción de casas pozo es proyecto adicional con su propia matriz).

### 3.2 Ingresos por lotes

- Precio promedio: USD 80.000-150.000/lote.
- 200 lotes × USD 110.000 promedio = USD 22.000.000 (sub-óptimo).
- Revisión: precios suben con avance de infraestructura → promedio efectivo USD 130k-150k.
- Ingreso total lotes: USD 26.000.000-30.000.000.

### 3.3 Ingresos por casas pozo (opcional)

- 50 casas de 200-300 m² a USD 1.500-2.000/m².
- Ingreso adicional: USD 18.000.000-25.000.000.

### 3.4 Financiación

- Auto-financiable (cobros lotes financian infra).
- Inversores ancla del 30-40%.
- Deuda bancaria limitada.

## 4. Cronograma

| Año | Hito |
|---|---|
| 0-1 | Compra tierra, master plan, permisos, lanzamiento pre-aprobación |
| 1-2 | Aprobación + movimiento de suelos + primera etapa infra |
| 2-4 | Comercialización + obra de infraestructura |
| 3-6 | Construcción de casas pozo + ventas escalonadas |
| 6-10 | Cierre + traspaso administración a la sociedad civil |

## 5. Comercialización

### 5.1 Etapas
- **Etapa 1** (sin infra): descuento 30-40% sobre precio final.
- **Etapa 2** (infra parcial): descuento 15-25%.
- **Etapa 3** (infra completa): precio lista.
- **Reventa de lotes**: a precio mercado.

### 5.2 Financiación al comprador
- 30% reserva + 70% cuotas en pesos ajustables (CAC o USD).
- Plazo 36-60 meses.
- Posibilidad de hipoteca privada del developer.

### 5.3 Casas pozo
- 30% boleto + cuotas + saldo entrega.
- Plazo construcción 12-18 meses.

## 6. Aspectos legales / tributarios

### 6.1 Forma jurídica
- Frecuente: sociedad civil del club o fideicomiso urbanístico.
- Reglamento del club + estatuto sociedad civil.
- Loteo + subdivisión catastral.

### 6.2 Aprobaciones
- Municipio: factibilidad urbanística + factibilidad servicios.
- Provincia (Buenos Aires): Decreto 9404/86 (clubes de campo), Decreto 27/98 (barrios cerrados) o régimen actualizado.
- Organismo provincial vivienda y urbanismo.
- ARBA + AGN: parcelamiento.
- ADA / ADL: agua + cloacas.
- ENRE / DPA: electricidad.
- Estudio de impacto ambiental.

### 6.3 Impuestos
- IVA 21% (loteo) — análisis con asesor, en algunos casos exento por venta de inmuebles.
- IIGG / Cedular vendedores.
- ITI o IIGG cedular 15% para personas humanas.
- Sellos provinciales.
- Inmobiliario provincial + municipal post-loteo.

### 6.4 UIF + AFIP
- Sujeto obligado: developer + inmobiliaria.
- DDC a cada comprador.

## 7. Riesgos y mitigación

| Riesgo | Mitigación |
|---|---|
| Aprobación municipal / provincial lenta | Equipo legal especializado + pre-negociación |
| Demanda suburbana cíclica | Etapas + flexibilidad |
| Infraestructura sobrepasa costos | Contingencia + redeterminación |
| Inundación / restricciones ambientales | Estudio hidráulico previo |
| Conflicto con vecinos / ONG | Diálogo + EIA |
| Inflación / FX | Cobros en USD o CAC |
| Reventa no se materializa | Restricciones contractuales a obras antes de plazo |
| Inseguridad regional | Inversión en seguridad + marketing del producto |

## 8. Indicadores objetivo

| KPI | Valor objetivo |
|---|---|
| TIR developer | 18-25% USD plazo largo |
| Plazo total | 7-10 años |
| % vendido año 5 | ≥ 60% |
| Margen bruto sobre tierra + infra | 25-35% |
| Costo infra por lote | USD 60.000-80.000 |

## 9. Particularidades del producto

### 9.1 Club de campo vs barrio cerrado
- Club de campo: área deportiva mayor, mayor superficie verde.
- Barrio cerrado: menos amenities, más densidad.
- Régimen legal: matiz por jurisdicción.

### 9.2 Restricciones edificables
- Reglamento define alturas, retiros, materiales, etc.
- Cuidan el valor del producto.

### 9.3 Expensas del club
- Una vez vendido: cuota mensual por mantenimiento.
- Importante para sostenibilidad post-developer.

## 10. Lecciones / pitfalls

- Subestimar costo de infraestructura (más caro que la tierra).
- Aprobación municipal optimista — planificar holgura.
- Vender muy rápido las primeras etapas con descuento mata margen.
- No mantener reserva para amenities → club no se completa.
- Conflictos con futuros vecinos / asociación civil.
- Olvidar EIA / impacto hidráulico.
- Confundir régimen club de campo con barrio cerrado.
- Comercializar con USD billete sin claridad sobre canal legal.

## 11. Reglas operativas para el chat

- **Estable:** estructura, hitos, lógica de loteo + casas.
- **🔴 Volátil:** precios USD, % avance, USD/m² casas → mercado al día.
- **Sensible:** aprobaciones urbanísticas y ambientales requieren expertise legal.
