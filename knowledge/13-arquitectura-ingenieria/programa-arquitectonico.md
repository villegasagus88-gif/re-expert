# Programa, anteproyecto, proyecto, dirección

## TL;DR
- Cuatro etapas del trabajo del arquitecto sobre un proyecto: **programa → anteproyecto → proyecto → dirección**.
- Cada una tiene productos, plazos y honorarios diferenciados.
- El programa lo define el **developer** (no el arquitecto). El arquitecto traduce.
- Encargo total típico = 8-12% del costo de obra. Sin dirección, 5-7%.

## 1. Programa arquitectónico (define el developer)

### 1.1 Qué es
- Documento que define **qué se quiere construir, para quién y con qué objetivos**.
- Es la traducción del modelo de negocio en pedido técnico.

### 1.2 Contenido típico
- Destino del edificio (vivienda, oficina, mixto).
- m² totales objetivo.
- Mix de tipologías (1amb, 2amb, 3amb, etc.).
- Calidad / categoría (estándar, media, alta, premium).
- Amenities deseados (piscina, gym, SUM, cochera).
- Restricciones del terreno (FOT, FOS, altura, retiros).
- Presupuesto target por m².
- Plazo objetivo.
- Cliente target / segmento.

### 1.3 Quién participa
- **Developer** (líder).
- **Arquitecto** (revisor técnico).
- **Comercial** (input de mercado).
- **Financiero** (validación de rentabilidad).

### 1.4 Errores comunes
- Programa sin análisis comercial → producto que no se vende.
- Programa sin restricciones urbanísticas → anteproyecto irrealizable.
- Programa "abierto" → fricción permanente con el arquitecto.

## 2. Anteproyecto

### 2.1 Qué entrega
- Idea arquitectónica + plantas, cortes, vistas básicas.
- Volumetría general.
- Mix de tipologías con superficies.
- Pre-cálculo de m² vendibles vs construidos.
- Estimación preliminar de costos.
- Resolución de **factibilidad urbanística** (verificación con el código).

### 2.2 Para qué sirve
- Tomar decisión GO/NO-GO del desarrollo.
- Presentar a inversores.
- Validar con autoridades urbanísticas (consulta previa).
- Base del estudio de factibilidad económico-financiera.

### 2.3 Duración típica
- 4-12 semanas según complejidad.

### 2.4 Honorarios típicos
- 15-25% del honorario total del proyecto.

## 3. Proyecto ejecutivo

### 3.1 Qué entrega
- **Planos municipales** (para permiso de obra).
- **Planos de obra**: planta, cortes, vistas, detalles constructivos.
- **Planos de instalaciones**: eléctricas, sanitarias, gas, calefacción, AC, ventilación.
- **Planos estructurales** (en coordinación con ingeniero).
- **Planos especiales**: ascensores, salas técnicas, terminaciones.
- **Pliego de especificaciones técnicas**.
- **Cómputo y presupuesto** detallado.
- **Memoria descriptiva** y memoria técnica.
- **Manual de mantenimiento** (en algunos casos).

### 3.2 Duración típica
- 12-30 semanas según complejidad.
- Para edificios complejos, se subdivide en etapas (proyecto preliminar / final).

### 3.3 Honorarios típicos
- 50-65% del honorario total del proyecto.

### 3.4 Resultado
- Documentación suficiente para **licitar la obra** y construirla sin sorpresas mayores.

## 4. Dirección de obra

### 4.1 Qué hace
- Supervisión técnica durante la construcción.
- Aprobación de muestras, materiales, sistemas.
- Resolución de dudas en obra.
- Aprobación de certificados de obra del constructor.
- Coordinación de subcontratistas.
- Recepciones parciales y final.

### 4.2 Modalidades
- **Dirección de obra clásica**: el arquitecto supervisa lo construido por una empresa.
- **Ejecución de obra**: el arquitecto/empresa la construye directamente.
- **Project management / gerenciamiento**: gestión integral del proyecto (más amplio).

### 4.3 Duración
- Igual al plazo de la obra (24-36 meses típico edificio).

### 4.4 Honorarios típicos
- 20-30% del honorario total del proyecto.
- Algunos cobran "porcentaje del avance" mensual.

## 5. Encargo total y aranceles

### 5.1 Tradicional (CPAU, CAPBA)
- Aranceles por escala progresiva (decrecientes con el monto).
- Bonificaciones habituales en mercado: 30-60% sobre el arancel oficial.

### 5.2 Honorarios típicos de mercado (no oficiales)
- Encargo completo (proyecto + dirección): **6-10% del costo de obra**.
- Solo proyecto: **3-5%**.
- Solo dirección: **3-5%**.

### 5.3 Aportes
- A CPAU / CAPBA: % adicional sobre honorario.
- Ingresos Brutos: alícuota provincial sobre honorario.
- IVA: 21% sobre honorario si el arquitecto es responsable inscripto.

## 6. Etapas del proyecto vs etapas del desarrollo

```
Desarrollo            →  Arquitectura
─────────────────────    ──────────────────────────
Definición del negocio   Programa
Factibilidad             Anteproyecto + factibilidad económica
Permiso de obra          Proyecto ejecutivo + planos municipales
Construcción             Dirección de obra
Final / posventa         Final de obra (DGROC) + manual mantenimiento
```

## 7. Roles paralelos al arquitecto

| Rol | Cuándo se suma |
|---|---|
| Ingeniero estructural | Anteproyecto avanzado / proyecto |
| Ingeniero electromecánico | Proyecto |
| Ingeniero sanitario | Proyecto |
| Ingeniero en gas | Proyecto |
| Especialista en incendios | Proyecto (depende escala) |
| Estudio de suelos (geotécnico) | Anteproyecto / inicio proyecto |
| Especialista BIM | Si se trabaja BIM (proyecto) |
| Tasador | Antes de la compra del terreno + control |
| Especialista en seguridad e higiene | Inicio de obra |

## 8. Errores comunes

- Empezar el proyecto sin programa cerrado.
- Saltar el anteproyecto y "tirar a planos de obra" — recipe for disaster.
- Cambiar el programa durante el proyecto → costos extra.
- Dirección de obra a cargo del propio constructor (conflicto de interés).
- Honorarios negociados muy bajos → calidad de proyecto pobre.
- No prever coordinación BIM cuando el proyecto es complejo.

## 9. Reglas operativas para el chat

- **Estable:** etapas, productos por etapa, plazos típicos, % de honorarios.
- **🔴 Volátil:** aranceles oficiales del año, % bonificaciones de mercado → CPAU/CAPBA.
- **Sensible:** decisión de contratar etapa por etapa o paquete completo → caso a caso.
- Recordar: **proyecto firmado por arquitecto matriculado** es obligatorio para permiso de obra.
