# Tecnología y PropTech

Módulo sobre cómo la tecnología transforma la cadena del real estate: data, IA, automatización, plataformas y nuevas estructuras.

## Archivos del módulo

- `panorama-proptech-ar.md` — segmentación del ecosistema PropTech argentino, principales actores, inversión.
- `ia-en-real-estate.md` — IA aplicada: AVM, lead scoring, generación de contenido, asistentes virtuales.
- `automatizacion-procesos.md` — RPA, workflows, integraciones API entre CRM/portales/ERP.
- `datos-y-fuentes.md` — open data, scraping, APIs, indicadores de mercado, BI.
- `tokenizacion-blockchain.md` — fracción digital de inmuebles, smart contracts, custodia.

- **Estable:** conceptos, arquitecturas, segmentación del ecosistema.
