# Seguros aplicados a Real Estate

Módulo sobre cobertura de riesgos en todas las etapas del proyecto: pre-obra, obra, posventa, operación.

## Archivos del módulo

- `marco-ssn.md` — Superintendencia de Seguros, marco normativo, productos.
- `trc-construccion.md` — Todo Riesgo Construcción, qué cubre y qué no.
- `responsabilidad-civil.md` — RC del developer + constructora + profesionales.
- `art-decreto-911.md` — Aseguradora de Riesgos del Trabajo + Decreto 911/96 (obra).
- `caucion-y-garantias.md` — Seguro de caución, garantías de adelanto, cumplimiento, vicios.
- `hogar-copropiedad.md` — Seguros para edificios terminados, hogar individual, consorcios.

## Reglas operativas

- **Estable:** tipos de seguro, marco SSN, coberturas estándar.
- **🔴 Volátil:** primas, condiciones particulares → cotizar con aseguradora + broker de seguros.
- **Sensible:** elección de pólizas requiere broker matriculado SSN.
