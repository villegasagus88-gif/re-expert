# Estructura de costos del proyecto

## TL;DR
- Costo total proyecto = **suelo + hard cost (construcción) + soft cost (honorarios, derechos, gerencia, marketing) + financieros + impuestos + contingencias + utilidad**.
- Rango orientativo edificio residencial AMBA medio: suelo 15-25%, hard cost 50-60%, soft cost 8-15%, contingencias 3-8%, comercialización 4-8%, utilidad del developer 15-25%.
- Conocer la estructura permite controlar y optimizar palancas.

## 1. Costo del suelo

### 1.1 Componentes
- Precio del terreno.
- Sellos, escritura, honorarios.
- Estudio de suelos y dominial (DD).
- Demolición si aplica.
- Compensaciones / aportes urbanísticos.

### 1.2 Regla
- **Suelo ≤ 15-25% del precio de venta** en proyecto rentable.
- Si suelo > 30%: alarma (cap rate exigente, márgenes ajustados).

## 2. Hard cost (costo duro de construcción)

### 2.1 Definición
- Todo lo que construye el edificio físicamente.
- Lo cuantifica el cómputo + APU.

### 2.2 Sub-rubros típicos
| Rubro | % aprox del hard cost |
|---|---|
| Movimiento de suelos + fundaciones | 8-15% |
| Estructura HºAº | 18-25% |
| Mampostería | 4-8% |
| Instalación eléctrica | 5-8% |
| Instalación sanitaria | 4-6% |
| Instalación de gas | 1-2% |
| HVAC | 4-12% |
| Aberturas | 6-10% |
| Revestimientos + pisos | 8-14% |
| Pintura | 2-4% |
| Ascensores | 3-6% |
| Terminaciones / cocinas / baños | 6-12% |
| Espacios comunes / paisajismo | 2-5% |

> 🔴 Porcentajes orientativos. Varían por sistema, categoría y geografía.

### 2.3 Peso en el total
- Hard cost: **50-60%** del costo total del proyecto.

## 3. Soft cost

### 3.1 Definición
- Costos no constructivos pero necesarios para llevar el proyecto a término.

### 3.2 Sub-rubros
- **Honorarios profesionales** (arquitecto 4-6%, ingenierías 1-3%, dirección 2-3%): 7-12% del hard cost.
- **Derechos municipales** (delineación, construcción, conexiones): 1-3% del hard cost.
- **Permisos / aprobaciones / habilitaciones**.
- **Gerenciamiento / project management**: 2-5% del costo total.
- **Asesoría legal / impositiva**.
- **Tasaciones / certificaciones**.

### 3.3 Peso en el total
- Soft cost: **8-15%** del costo total.

## 4. Comercialización / marketing

### 4.1 Componentes
- Comisión broker (3-4% sobre venta + IVA).
- Marketing digital (Meta, Google, portales): 0.5-2% sobre venta.
- Stand / showroom / departamento muestra.
- Material gráfico / renders / video.
- Eventos de lanzamiento.

### 4.2 Peso
- Comercialización total: **4-8%** del precio de venta.

## 5. Costos financieros

### 5.1 Componentes
- Intereses de crédito puente / construcción.
- Costo de capital (oportunidad).
- Comisiones bancarias / estructuración.
- Garantías.

### 5.2 Variable según
- Mix equity vs deuda.
- Plazo de ejecución.
- Tasas vigentes (UVA, BADLAR, USD).

### 5.3 En AR
- Crédito a construcción residencial limitado.
- Mayoría se hace con equity + pre-venta en pozo.
- Costo financiero efectivo: alto por inflación + dolarización.

## 6. Impuestos

### 6.1 Sobre la construcción
- IVA (10.5% vivienda / 21% otros).
- IIBB (1-3.5%).
- Ganancias del developer (proyecto en sí).
- Sellos en contratos.

### 6.2 Sobre la venta
- IIBB.
- Sellos.
- Ganancias / Cedular (vendedor según caso).
- Bienes Personales (tenencia previa a venta).

### 6.3 Peso
- Total impuestos sobre el proyecto: **15-25%** según estructura.

## 7. Contingencias / imprevistos

### 7.1 % típico
- Anteproyecto: 10-15% sobre directos.
- Proyecto ejecutivo: 5-10%.
- Obra avanzada: 3-5%.

### 7.2 Por qué
- Cambios de diseño en marcha.
- Sorpresas en obra (suelo, vecino, clima).
- Inflación intra-período.
- Cambios regulatorios.

> Ver `./contingencias-imprevistos.md`.

## 8. Utilidad del developer

### 8.1 Margen objetivo
- Residencial estándar AR: **15-25%** sobre costos totales.
- Premium: 20-30%.
- BTR / renta: TIR objetivo 12-18% USD.

### 8.2 Distribución
- Si hay socios / inversores: waterfall (ver `../06-financiero/waterfall-inversores.md`).
- GP / promote / carried interest.

## 9. Tabla resumen (proyecto residencial AMBA medio)

| Concepto | % sobre venta |
|---|---|
| Suelo | 15-25% |
| Hard cost | 35-45% |
| Soft cost | 5-10% |
| Comercialización | 4-8% |
| Financieros | 2-5% |
| Impuestos (proyecto) | 5-10% |
| Contingencias | 3-5% |
| Utilidad developer | 15-25% |
| **Total venta** | **100%** |

> 🔴 Cifras orientativas; cada proyecto se modela específicamente.

## 10. Palancas para optimizar

- **Suelo**: comprar bien (timing + zona en crecimiento).
- **Hard cost**: BIM + coordinación + escala + acopio.
- **Soft cost**: equipo experimentado, evitar re-trabajos.
- **Comercialización**: pre-venta agresiva en pozo (financia obra).
- **Financieros**: optimizar estructura capital, vender rápido.
- **Impuestos**: planificación + figura adecuada (fideicomiso, SA, SAS).

## 11. Errores comunes

- Modelar el proyecto solo con "hard cost + 30%".
- Subestimar comercialización.
- No cargar impuestos completos en proyecto.
- Olvidar costos financieros del capital del propio developer.
- No prever contingencias suficientes.

## 12. Reglas operativas para el chat

- **Estable:** estructura conceptual, rubros, palancas.
- **🔴 Volátil:** % específicos por proyecto, precios, tasas → modelo financiero específico.
- **Sensible:** todo modelo definitivo lo firma profesional + auditor.
- Si el usuario pregunta "¿cuánto gana el developer?": rango 15-25% es referencia, depende mucho de cada proyecto.
