# Costos y presupuesto

Núcleo del control económico del proyecto: cómo se arma, controla y proyecta el presupuesto de obra desde anteproyecto hasta final de obra.

## Archivos del módulo

- `analisis-precios-unitarios.md` — APU: cómo se compone el precio de cada ítem (materiales, mano obra, equipos, gastos generales, utilidad).
- `estructura-costos.md` — costos directos, indirectos, gerenciamiento, contingencias, financieros.
- `curva-s.md` — distribución temporal del gasto vs avance; flujo de caja de obra.
- `indices-costo.md` — ICC INDEC, CAC, IPC, UVA, USD; ajustes y redeterminación.
- `contingencias-imprevistos.md` — % típicos por etapa, manejo de imprevistos, change orders.
- `control-presupuestario.md` — desvíos, valor ganado (earned value), reportes, alertas.

## Reglas operativas

- **Estable:** metodología, estructura, conceptos.
- **🔴 Volátil:** valores ARS/USD, índices vigentes, % por ítem → siempre redirigir a fuentes (INDEC, CAC, cotizaciones de mercado).
- **Sensible:** todo presupuesto definitivo lo firma profesional + cómputo independiente.
