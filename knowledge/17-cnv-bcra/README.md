# Mercado de capitales (CNV) + BCRA en RE

- `vehiculos-cnv-re.md` — FCI cerrados, fideicomisos financieros, ON, CEDEARs, mapeo aplicado a RE.
- `oferta-publica.md` — requisitos, prospectos, due diligence, agentes intermediarios.
- `cepo-cambiario.md` — BCRA, mercado cambiario, MULC, MEP, CCL, restricciones operativas.

- **Estable:** estructura institucional, vehículos, conceptos.
