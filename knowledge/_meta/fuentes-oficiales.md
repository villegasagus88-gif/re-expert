# Fuentes oficiales y profesionales — Argentina

Lista única y consolidada de fuentes confiables. **Toda información del KB
debe poder rastrearse a una fuente de esta lista** (o equivalente con la misma
calidad institucional).

## TL;DR
- Para macro/precios/inflación → **BCRA + INDEC**.
- Para impuestos → **AFIP (nación), ARBA (PBA), AGIP (CABA)**.
- Para normativa → **Boletín Oficial (BORA) + InfoLEG**.
- Para construcción → **CAC, CPAU, SCA, IRAM**.
- Para sociedades / fideicomisos → **IGJ (CABA), Dirección de Personas Jurídicas (cada provincia)**.

## 8. Mercado / data inmobiliaria (con criterio)

> Fuentes de mercado privadas: usar sólo para **rangos** y **tendencias**,
> nunca para "tasaciones oficiales". Marcar siempre `confidence: media`.

- Reporte Inmobiliario (reporteinmobiliario.com)
- ZonaProp / Argenprop / MercadoLibre Inmuebles → datos de listados (no de cierres)
- Colegios de Escribanos (CABA: colegio-escribanos.org.ar) → escrituración mensual
- Banco Ciudad — informe mercado inmobiliario (cuando se publica)

## 9. Académico

- **CEDLAS** (UNLP) — economía aplicada
- **Fundación Mediterránea / IERAL** — análisis fiscal y macro
- **FIEL** — política económica
- **IIE-UCA** — economía y desarrollo
- Papers indexados en SciELO Argentina

---

## ❌ Lo que NO usamos

- Blogs personales sin atribución.
- Hilos de Twitter/X / posts de LinkedIn como fuente primaria.
- Notas periodísticas sin link al dato original.
- Captura de Telegram, WhatsApp, foros.
- "Fuentes del mercado" sin nombre.
- Cualquier IA generativa como fuente final (sí como punto de partida, pero hay que verificar).
